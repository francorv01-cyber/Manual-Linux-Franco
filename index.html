Manual de Comandos Linux

CHMOD

Cambia permisos de archivos y directorios.

Opciones

  • chmod 644
  • chmod 755
  • chmod u+x

Ejemplos

chmod 644 documento.txt
chmod 755 script.sh
chmod u+x script.sh

WC

Cuenta líneas, palabras y caracteres.

Opciones

  • -l
  • -w
  • -c

Ejemplos

wc archivo.txt
wc -l archivo.txt
wc -w archivo.txt

DF

Muestra el espacio disponible en los sistemas de archivos.

Opciones

  • -h
  • -T

Ejemplos

df -h
df -T

DU

Muestra el tamaño de directorios y archivos.

Opciones

  • -h
  • -s

Ejemplos

du -h carpeta/
du -sh carpeta/
